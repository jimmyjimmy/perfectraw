========================================================================
    BIBLIOTECA DE VÍNCULOS DINÁMICOS: Información general del proyecto 
                                      pruDLL
========================================================================

AppWizard ha creado este archivo DLL pruDLL.  

Este archivo incluye un resumen acerca del contenido de los archivos que
constituyen su aplicación pruDLL.


pruDLL.vcproj
    Éste es el archivo de proyecto principal para los proyectos de VC++ 
    generados mediante un Asistente para aplicaciones. 
    Contiene información acerca de la versión de Visual C++ con la que 
    se generó el archivo, así como información acerca de las plataformas, 
    configuraciones y características del proyecto seleccionadas en el
    Asistente para aplicaciones.

pruDLL.cpp
    Éste es el archivo de código fuente DLL principal.

	Cuando se crea este archivo DLL, no exporta símbolos. Por lo 
        tanto, no creará un archivo .lib cuando se genere. Si desea 
        que este proyecto sea una dependencia de otro proyecto, es 
        preciso agregar código para exportar algunos símbolos del 
        archivo DLL para crear una biblioteca de exportación, o establecer 
        la propiedad Omitir biblioteca de entrada en Sí en la página de 
        propiedades General en la carpeta Vinculador del cuadro de diálogo
        Páginas de propiedades del proyecto.
